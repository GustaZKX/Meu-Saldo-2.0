# **App Name**: Meu Saldo IA

## Core Features:

- Dashboard Financeiro: Visão geral das finanças do mês, incluindo resumo de receitas, despesas e saldo, juntamente com um calendário interativo de vencimentos.
- Gerenciamento de Ganhos e Despesas: Adicionar, editar e excluir registros de ganhos e despesas, categorizando-os e marcando despesas como pagas.
- Contadora IA: Insights financeiros automatizados com base nos dados inseridos, incluindo sugestões de limites de gastos e acompanhamento do progresso das metas.
- Análise Inteligente de Limite de Gastos: O sistema usa uma 'tool' para analisar os padrões de gastos do usuário e fornecer sugestões personalizadas sobre os limites de gastos diários e semanais, ajudando a manter o orçamento sob controle.
- Planejamento de Metas: Definir metas financeiras, calcular o compromisso mensal necessário para atingi-las e acompanhar o progresso ao longo do tempo.
- Relatórios: Visualização gráfica de relatórios de gastos e ganhos por categoria, permitindo o filtro por mês.
- Personalização de Cores: Permitir que o usuário personalize as cores do aplicativo, especialmente para os gráficos de relatório.

## Style Guidelines:

- Primary color: Dark blue (#1A237E). A professional tone is struck with this deep, trustworthy hue, commonly associated with finance and reliability.
- Background color: Very light gray (#F5F5F5). To provide contrast with the content-rich cards and elements on the page, this backdrop creates a subtle canvas that is aesthetically and functionally complementary.
- Accent color: Light blue (#0277BD). To guide users to key actions and create a feeling of trust, it serves as an interactive contrast, without stealing the focus from financial details.
- Body and headline font: 'Inter' sans-serif for clear readability and a modern, neutral aesthetic.
- Ícones claros e intuitivos para representar diferentes categorias de gastos e funcionalidades do aplicativo.
- Layout responsivo e adaptável a diferentes tamanhos de tela, com design focado em acessibilidade e usabilidade.
- Animações sutis para feedback visual em interações, como ao adicionar uma nova transação ou salvar uma meta.
